**Walmart Sales Data Analysis Project**
This project is an end-to-end data analysis solution built to extract key business insights from Walmart sales data. It combines Python for data processing and cleaning with SQL for advanced querying and analytics.

**Overview**
The project demonstrates how to:
Download and prepare real sales data using the Kaggle API
Clean, transform, and enhance data with Pandas
Load data into MySQL and PostgreSQL databases
Use SQL to answer business questions such as revenue trends, top categories, and branch performance

**Tools & Technologies**
Languages: Python, SQL
Databases: MySQL, PostgreSQL
Libraries: pandas, numpy, sqlalchemy, mysql-connector-python, psycopg2
Environment: Visual Studio Code
Data Source: Walmart Sales Dataset (Kaggle)
